HTML

  • HTML is one of the three cornerstone languages in web development
  • Text, image, header, and list elements are most commonly used to add content to HTML webpages.
  • Meta-realated, content divisionand semantic are all elements that divide and describe a webpage.
  • All these elements are use together to structure a webpage.
  • HTML is essentially used to build a frame for a webpage.
  • HTML has many different elements that play an important role.
  • Each element has it own role.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS helps style an HTML webpage.

Git

  • Git is a version control system.
  • Git helps you manage each version of your projects.
  • Git commands always start with git.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Perfect for tracking changes on your personal computer.
  • Used to create or add code to repositorys.
  • GitHub is used to manage projects in repositorys and to collaborate with others.
  • GitHub is one of many Git based respository hosting services.
